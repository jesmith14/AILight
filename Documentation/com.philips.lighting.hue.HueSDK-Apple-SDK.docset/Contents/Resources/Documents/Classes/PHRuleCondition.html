  address

local address: path to a light resource, a group resource, or any other bridge resource

@property (nonatomic, strong) NSString *address

Discussion

web address: URL of the Portal, an intranet server for a RESTful call using the given method to send the given body. The server address http://www.meethue.com/ addresses the Hue portal. Examples http://www.meethue.com/api/events http://192.168.168.1/api/sendtodropbox The rule is deactivated if the server does not respond (error code 7). The lost connection of http://www.meethue.com does not cause deactivation.
